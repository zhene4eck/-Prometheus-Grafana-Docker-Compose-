# Развертывание системы мониторинга Linux-сервера

## Описание

В рамках проекта была реализована система мониторинга Linux-сервера для централизованного сбора, хранения и визуализации системных метрик.

Для решения задачи развернут стек **Prometheus + Grafana + Node Exporter** в контейнерах Docker с использованием Docker Compose.

Система обеспечивает непрерывный сбор метрик состояния сервера и предоставляет веб-интерфейс для их анализа в режиме реального времени.

---

| Цель | Организовать мониторинг Linux-сервера |
|----------|:----------|
| Задача | <ol><li>Контролировать загрузку процессора.</li><li>Отслеживать использование оперативной памяти.</li><li>Контролировать использование дисковой подсистемы.</li><li>Анализировать сетевую активность.</li><li>Получать информацию о состоянии файловых систем.</li><li>Визуализировать показатели в едином интерфейсе.</li></ol> |
| Требовнаие | Обеспечить простое развертывание и возможность дальнейшего масштабирования решения. |


---

## Реализованное решение

Для выполнения поставленной задачи был развернут следующий стек сервисов:

- **Node Exporter** — сбор системных метрик Linux;
- **Prometheus** — сбор и хранение метрик;
- **Grafana** — визуализация данных и построение дашбордов;
- **Docker Compose** — оркестрация контейнеров.

Все сервисы запускаются одной командой и взаимодействуют внутри общей Docker-сети.

## Результат

В результате внедрения была развернута система мониторинга Linux-сервера, обеспечивающая автоматический сбор и визуализацию системных метрик.

Полученное решение позволяет оперативно контролировать состояние сервера, анализировать использование вычислительных ресурсов и использовать дашборды Grafana для мониторинга ключевых показателей производительности.

---

## Возможности дальнейшего развития

Проект может быть расширен за счет:

- подключения Alertmanager для отправки уведомлений;
- мониторинга нескольких серверов;
- интеграции cAdvisor для мониторинга Docker-контейнеров;
- подключения Loki для централизованного хранения логов;
- разработки собственных дашбордов и правил оповещения.
